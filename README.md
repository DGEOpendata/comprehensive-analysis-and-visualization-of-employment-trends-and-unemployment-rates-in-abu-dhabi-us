# Employment Trends and Unemployment Rates Analysis

This repository contains code and resources for analyzing the employment trends and unemployment rates in Abu Dhabi for the year 2022. The dataset includes detailed demographic, sectoral, and gender-based employment insights.

## Features:
- Data Cleaning and Preparation
- Exploratory Data Analysis (EDA) with visualizations
- Predictive modeling for unemployment trends

## Requirements:
- Python 3.8+
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Installation:
bash
pip install pandas matplotlib seaborn scikit-learn


## Run the Analysis:
1. Clone the repository.
2. Place the dataset in the root directory.
3. Run `analysis.py` to generate insights and predictions.

## Outputs:
- Visualization of employment trends by sector and gender.
- Predictive modeling results for unemployment rates.
